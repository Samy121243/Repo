# University Management System – UI/UX Design

## Project Overview
A digital platform designed to support **students, faculty, and administrators** in managing academic and administrative operations efficiently.  
This project focused on delivering a **complete UI/UX redesign**, covering **Splash / Loading Screen**, **Login**, **Dashboard**, and core management modules (**Departments, Students, Teachers**) with a focus on **usability, clarity, and user engagement**.

## Problem Statement
Traditional university systems often suffer from:
- Confusing loading and login flows
- Overcrowded dashboards and menus
- Poor feedback and error handling
- Weak data visualization and high cognitive load

## Goals
- Create an engaging **Splash / Loading experience**
- Simplify login with **Face ID** and minimal form inputs
- Reduce cognitive load and simplify navigation
- Improve discoverability of core functions
- Deliver fast, clear, and error-resistant workflows
- Establish a strong **system identity**

## Key Features & Improvements
- **Splash Screen:** Animated progress bar + faculty building image for clarity and engagement  
- **Login:** Minimal design, Face ID support, clear feedback  
- **Dashboard:** Simplified layout with 6–7 core functions, improved discoverability  
- **Departments & Students Screens:** Clear data visualization, fast actions, live feedback  
- **Teachers Screen:** Advanced administrative controls, context menus, consistent UI
